# 🚀 PHireScript

Bem-vindo ao **PHireScript**.

Uma linguagem projetada para combinar a simplicidade do PHP com recursos modernos de tipagem, expressividade e uma sintaxe mais fluida.

---

## ✨ Filosofia

PHireScript busca:

- Reduzir ruído sintático
- Tornar o código mais legível
- Melhorar a segurança através de tipagem
- Manter a familiaridade do ecossistema PHP
